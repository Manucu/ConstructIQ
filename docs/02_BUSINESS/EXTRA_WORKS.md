# Purpose

This document defines how additional construction works (Extra Works) are managed inside ConstructIQ.

Extra Works represent any activity requested after the original project scope has been approved.

These works must remain completely traceable from the initial request until the final invoice.

---

# Scope

This document applies to:

- additional construction works
- client requests
- design changes
- unforeseen site conditions
- quantity increases
- additional labor
- additional materials

---

# Definitions

## Original Scope

The work initially agreed between the company and the client.

## Extra Work

Any work performed outside the original contract.

## Change Order

A formal approval allowing an Extra Work to become part of the project.

- labor value

# Permissions

## Site Engineer

Can

- create
- edit before approval
- upload photos
- register hours
- register materials

Cannot

- define selling price
- approve
- invoice

---

## Project Manager

Can

- review
- request corrections
- approve operational information

Cannot

- define final selling price

---

## Owner

Can

- define selling price
- approve
- reject
- archive
- invoice

---

## Accountant

Can

- generate invoice
- manage VAT
- update accounting status

---

## Client

Can

- review quotation
- approve or reject
- download approved documents
