# ConstructIQ

**Building Intelligence into Every Project**

## 1. Project Identity

ConstructIQ este o platformă software pentru managementul inteligent al proiectelor de construcții civile.

Produsul urmărește digitalizarea planificării, monitorizării și raportării proiectelor de construcții.

## 2. Workstreams

Proiectul este organizat în trei fluxuri de lucru:

### Product
Definește ce construim:
- viziune
- cerințe
- backlog
- wireframe-uri
- use cases

### Software
Definește cum construim:
- backend Spring Boot
- frontend React
- PostgreSQL
- API REST
- autentificare
- testare

### Thesis
Definește cum explicăm proiectul:
- capitole licență
- diagrame
- capturi de ecran
- prezentare finală
