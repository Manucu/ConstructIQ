# Purpose

The Budget System defines how project budgets are planned, monitored and controlled throughout the lifecycle of a construction project.

Its objective is to provide complete visibility over planned budgets, actual costs and remaining funds while ensuring that financial information is always based on approved data.

---

# Scope

The Budget System manages:

- Project Budget
- Stage Budgets
- Labor Budget
- Material Budget
- Equipment Budget
- Transport Budget
- Other Costs
- Extra Works
- Budget Consumption

---

# Budget Philosophy

Every project starts with an estimated budget.

During execution the budget evolves using approved project data.

ConstructIQ never edits historical costs.

Every budget update is fully traceable.

---

# Budget Structure

Every Project contains one Budget.

The Project Budget is divided into Stage Budgets.

Each Stage Budget contains:

- Labor
- Materials
- Equipment
- Transport
- Other Costs
- Extra Works

---
